Horoskop Projekt

Ein kleines Java-Projekt zur Berechnung von Sternzeichen, chinesischem Tierkreiszeichen und Alter in Tagen.
Der Benutzer gibt Tag, Monat und Jahr ein und das Programm zeigt verschiedene Informationen an.

Funktionen
Bestimmung des westlichen Sternzeichens
Bestimmung des chinesischen Tierkreiszeichens
Berechnung des Alters in Tagen
Eingabe über die Konsole
Projektstruktur

Das Projekt besteht aus mehreren Klassen:

WesternZodiac

Berechnet das westliche Sternzeichen:

Widder
Stier
Zwillinge
Krebs
Löwe
Jungfrau
Waage
Skorpion
Schütze
Steinbock
Wassermann
Fische
ChineseZodiac

Berechnet das chinesische Tierkreiszeichen:

Affe
Hahn
Hund
Schwein
Ratte
Ochse
Tiger
Hase
Drache
Schlange
Pferd
Ziege
AltersRecner

Berechnet das Alter in Tagen und prüft Schaltjahre.

Goroskop

Hauptklasse mit Benutzereingabe und Ausgabe.

Beispiel
Tag: 15
Monat: 8
Jahr: 2005

Sternzeichen: Löwe
Chinesisches Tier: Hahn
Alter in Tagen: 7560
Verwendete Technologien
Java
Eclipse IDE
